# Screener 2025

Український інструмент для створення скріншотів з вбудованим редактором.

## Можливості
- Виділення області екрану
- Стрілки, текст, рамки, блюр
- Копіювання в буфер обміну
- Завантаження в Google Drive

## Безпека
- Програма **не має повного доступу до Google Drive**
- Використовується лише дозвіл `drive.file`
- Доступ надається **тільки до файлів, створених програмою**
- Дані нікуди не передаються, окрім Google Drive

## Завантаження
Перейдіть у розділ **Releases** та завантажте останню версію для Windows.
